# Ubuntu Tracks
Ubuntu Tracks is a next-generation event management platform(EMP) built to bridge the gap between event organizers and their audiences. It provides a suite of interactive tools for registering for an event, discover upcoming events, and post-event data harvesting. The platform focuses on maximizing attendee satisfaction while giving organizers the data-driven insights they need to grow. 

## Getting Started

First, Clone this repository:

```bash
git clone "https://github.com/"
```
after cloning, Make sure your in the correct directory

```bash
cd Vento_Tracks
```
then, Install all dependencies in the app
```bash
bun install
#or
npm install
```
Now, You can run the development server

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
